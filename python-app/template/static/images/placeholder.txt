# Images Directory

This directory contains all the images used in the application.

## Directory Structure:
- /gallery/ - Images for the photo gallery
- /portfolio/ - Images for portfolio projects
- hero-bg.jpg - Hero background image
- profile.jpg - Profile picture for about page

## Image Requirements:
- Gallery images: 800x600px recommended
- Portfolio images: 600x400px recommended
- Hero background: 1920x1080px recommended
- Profile image: 400x400px recommended

## Supported Formats:
- JPEG (.jpg, .jpeg)
- PNG (.png)
- WebP (.webp)
- SVG (.svg)

## Notes:
- All images should be optimized for web use
- Use descriptive filenames
- Add alt text for accessibility
- Consider using WebP format for better compression
