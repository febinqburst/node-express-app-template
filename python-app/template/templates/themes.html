Theme Customization

Want to Create Your Own Theme?

Our theming system is built with extensibility in mind. You can easily create custom themes by:

  • Adding new CSS files in the themes directory
  • Following our CSS variable structure
  • Updating the theme configuration in the application
  • Testing across all components and pages

Each theme consists of color variables, gradients, and component-specific styling that maintains consistency across the entire application.

Example Theme Structure

/* Custom Theme */
.theme-custom {
    background: linear-gradient(135deg, #color1, #color2);
    color: #text-color;
}

.theme-custom .navbar {
    background: rgba(255, 255, 255, 0.95);
}

.theme-custom .btn-primary {
    background: #primary-color;
    color: white;
}
